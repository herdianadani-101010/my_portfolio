🌊 Flood Early Warning System (EWS)

Low-cost flood detection system using 4 float switches and ESP8266 (Wemos D1), integrated with Telegram and WhatsApp bot for real-time alerting.

  • 4 level detection: Aman, Siaga 3, Siaga 2, Siaga 1
  • Realtime notification (Telegram & WhatsApp)
  • Data logging to Google Sheets
  • Cost implementation ± Rp 120.000

Working Principle:

  • Float switch triggers based on water level
  • ESP8266 reads digital input
  • Data sent to cloud + notification triggered
